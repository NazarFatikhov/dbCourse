1. Создать нескольких пользователей:
	user1 с аутентификацией Windows (для этого вам придется предварительно создать такого пользовать windows)
   	user2 с аутентификацией SQL Server
	user3 без возможности аутентификации
Проверить возможность логина для всех созданных пользователей.
2. Удалить пользователя user2. Для него предоставить право создания БД.
3. В БД model создать таблицу.
4. Проверить, что при создании новых БД эта таблица булет в них пристутствовать.
5. Изменить БД - сменить владельца на нового пользователя user4
6. Создать новую базу данных, состоящую из двух файловых групп, одного журнала. В первичной группе описать 2 файла, разбросанные по различным дискам (один из них имеет размер 100 МБ, рост до 100 МБ, с шагом 20 МБ). Две вторичные группы: каждая состоит из двух файлов (первый из которых растет с шагом 10 МБ, размер не ограничен, второй имеет постоянный размер). Вторая группа является группой по умолчанию (DEFAULT). Файл журнала растет на 30%.
7. Внутри БД создать новую схему. Создать в ней таблицу с одним полем id(int).
8. Сделать выгрузку БД. 
9. Загрузить эту выгрузку в новую БД
10. Дать все права в новой БД пользователю user1 и user4.
11. Удалить права для одного из пользователей на новую БД (user4)
12. Дать права пользователю user3 на выборку (SELECT), но не давать на (CREATE TABLE). Проверить, что всё корректно работает.
